<head>	 	
</head> : <body> 본문에 쓰인 데이터를 브라우저 창에 출력될 때
  	  	  시각적인 효과(색깔, 가운데 정렬, 글씨 테두리 색..등등) 
	 	  를 설정할때 쓰이는 태그문단, CSS작업을 여기서 하게됩니다.		 


<body>
</body> : 본문이 쓰이는 태그문단, 기능이 만들어지는 태그문단이다.

 
블록레벨    : 다른 태그 문단을 옆에 붙여서 작성 할 수 가 없습니다. 수직적(다른 줄에 차곡차곡) 으로 쌓입니다.
인라인 레벨 : 다른 태그 문단을 옆에 붙여서 작성 할 수 가 있습니다. 수평적(같은 줄에 차곡차곡) 으로 쌓입니다.


<p>
</p>        : 블록레벨의 태그문단으로서, 여기안에 쓰이는 문단은 띄어쓰기는 단 한번만 적용된다(여러개를 작성해도)
	           엔터 줄바꿈도 적용이 안된다(<br>로 엔터와 동일한 효과를 줄수 있다,<br>작성한 만큼 효과가 부여됨)


<pre>
</pre> : 블록레벨의 태그문단으로서, <p> 문단의 개선판 같은 느낌 여기서는 띄어쓰기 줄 바꿈 모두 정상적으로 적용됨


<div>
</div> : <p>태그랑 블록레벨인것도 똑같고 기능이 비슷합니다, 단 수업중에는 <div>만 사용, 성능상 이점이 있는거 같습니다
		   앞으로 웬만하면 이것만 이용하시면 됩니다.


엔티티코드(라고 읽고 그냥 특수기호 입니다, 문단 태그안에 문자 입력 하실 때 사용하시면 괄호안에 있는 기호가 입력됩니다)		
    작다(less than)                  &lt;  ( < )  
    작거나 같다(less than equal)     &le;  ( <= ) 
    크다(greater than)               &gt;  ( > )  
    크거나 같다(greater than equal)  &ge;  ( >= )  
    앰퍼센드                         &amp; ( & ) 
    공백(띄어쓰기)                   &nbsp;
