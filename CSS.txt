1. CSS 문단을 보기좋게 꾸며주는 역할을한다
   1-1. <head>에 <style></style>안에 설정해주거나 
   1-2. 따로 파일을 만들어 설정해주거나(실무에서는 이방식 추천)
   1-3. <body></body>안에 문단에 설정값을 주거나

1-2. <link rel="stylesheet" href="경로 설정">
	jdn이랑 비슷하게 설정된값이 있는 파일로 연결해줘야 한다.

태그문자 속성에 class 값을 따로 설정해주면 속성을 부여할수있는데
호출할 때 .class(별명)로 호출해 <style>태그문안에 속성을 부여해주면된다
추천하는 방법은 아니지만 head문 말고 body문 안에 <style>태그문을 생성해도 
적용된다.

id도 class 처럼 체감상 똑같은 역할을 하지만 호출할때는 #id(별명)로 호출한다

자식선택자는 태그수가 더 많아지고 종속된 태그가 더 많아 질수록 
안쪽에 있는 태그 속성을 부여해주고 싶을때 쓰는 기능이다.
