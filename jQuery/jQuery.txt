jQuery란?
하나의 새로운 기능(도구)이다
일반객체(document)를 jQuery객체화를 시켜야만 jQuery 메소드를 사용할 수 있다.


jQuery 사용 방법
$('jQuery객체화 할 대상') 이 형식으로 jQuery객체화를 시킨다.
$('#id값'), $('.class값'), $('태그종류')
같은위치 모든 태그(ex, ~ li) 아래있는 태그(ex, > li) 인접한 태그(ex, + li)
CSS랑 기능은 비슷한데 
jQuery에서만 쓰이는 특별한 메소드가 있다 
() 안에 기입해주는 데이터에 따라 선택지를 구별해준다.
다 외울 필요보다는 쓰다보면서 익히면 될 것 같다.
.find()          : 하위 태그 중 원하는 태그 
.children()      : 모든 자식 태그
.parents()       : 모든 부모 태그
.parent()        : (가장)인접한 부모 태그
.siblings()      : 모든 형제 태그
.next()          : 인접 형제 태그
.nextAll()       : 모든 형제 태그
.prev            : 이전(부모) 형제 태그
.prev            : 이전(부모) 모든 형제 태그
.closest         : 가장 가까운 부모 태그
input[type=text] : 텍스트 타입으로 지정하기
: type명         : 타입명으로 지정하기
: 상태명         : 상태명으로 지정하기 (ex. focus, checked, selected...)

이와 더불어 CSS나 text를 기입해준다거나 attribute을 부여 또는 property 조차도 모두 조작 가능하다 
id명(or class명).text() : 태그 안 에 있는 텍스트를 변경해 줄 수 있다
                .html() : 태그 안 에 있는 텍스트를 html 기능을 구현하면서 변경해 줄 수 있다
                .css()  : 태그 안 에 있는 데이터의 CSS변화를 줄 수 있다 ex. .css('color', 'black')
                .attr() : 태그 안 에 있는 속성 값을 부여해주거나 수정이 가능하다
                .prop() : 태그 안 에 있는 속성 값을 부여해주거나 수정이 가능하다(attr()랑 유사하다), 자바스크립트 안에서 객체에 속성변화를 준다
                          (.attr이랑 똑같이 작용하나 이미지 크기 조절할 때 px 단위 붙이면 정상 작동 안함)

태그 자체를 추가하거나 삭제하는것도 가능하다.
부모.prepand(<자식></자식>) : 첫 번째 자식으로 추가하기
    .append(<자식></자식>)  : 마지막 자식으로 추가하기, 압도적으로 많이 쓴다.                     
자식.prependTo(부모)        : 첫 번째 자식으로 추가하기
자식.appendTo(부모)         : 마지막 자식으로 추가하기
                                $(<li>).attr('id', 'hamburger').text('햄버거').appendTo(menu);
                                이런식으로 응용도 가능함
$('태그아이디명'(or'클래스명')).remove() : 특정 요소를 제거 가능하다
$('태그아이디명'(or'클래스명')).empty()  : 모든 요소를 제거 가능하다, 다만 태그 자체가 삭제 되는건 아니다.

jQuery의 load 이벤트란? 
공식적으로는 <div>태그를 <script>문 위에다 작성해야지만 순서가 맞아서 데이터를 
읽을수 있지만(컴퓨터가 정상적으로 인식하지만) <script> 문안에 load 이벤트를 사용하고 
작성 시 <div> <script> 순서 상관없이 모든요소를 인식할 수 있다. 

=>  <script>
    $(function(){
        $('jQuery객체화 할 대상')        
    })
    </script>

this 반복문 함수에서는 계속 값을 대입 해주는 역할을 하였고
이벤트문에서는 동작하는 이벤트를 인식해주는 역할을 했다.
